INTERNET HISTORY WEBSITE

Overview:
- This website is a comprehensive, multi-page educational website that explores the origins of the internet, key computing pioneers, and essential web protocols. The site serves as an accessible knowledge base, breaking down complex networking concepts and historical milestones through well-structured HTML and styled with external CSS.

Key Features:
- Multi-Page Navigation: Features a consistent header navigation menu that seamlessly links between the Intro, Internet Pioneers, HTTP, Wiki, Terminology, and References pages.
- In-Depth Historical Content: Details the evolution of the internet, starting from the ARPAnet in the 1950s up to the development of the World Wide Web and early browsers like Mosaic and Lynx.
- Pioneer Biographies: Showcases the lives and contributions of 10 key figures in computer science, including Vannevar Bush, Tim Berners-Lee, Vint Cerf, and Bob Kahn.
- Protocol Explanations: Provides a step-by-step breakdown of how a simple HTML page is requested and served using HTTP and HTTPS protocols, supplemented by a custom HTTP diagram.
- Wiki Mechanics: Explains the collaborative nature of Wikis, their software engines, their origins dating back to Ward Cunningham in 1994, and how they differ from Wikipedia.
- Terminology Dictionary: Utilizes HTML definition lists to create a structured dictionary of essential web acronyms and concepts, such as TCP/IP, GUI, HCI, and SFTP.
- Printable PDFs: Every page features a footer link allowing users to easily download a printable PDF version of the current page's content.
- Academic Citations: Includes a dedicated References page that meticulously attributes all textual information and media used throughout the site.

Tech Stack:
- Markup: Semantic HTML5 to ensure accessible and well-organized content delivery.
- Styling: External CSS stylesheets linked directly to each individual page (index_css.css, pioneers_css.css, etc.) to maintain modular and organized visual formatting.
