Internet
The Internet is a global system of interconnected worldwide networks [1]. It is the base, the foundation for online applications.
World Wide Web
World Wide Web (WWW) is a global system of interlinked hypertext documents on the Internet [1]. It is a universe of accessible information and all of the information, the interlinked hypertext documents, is connected by the Internet [1], [2].
Hypertext
Hypertext is a text which contains links (hyperlinks) to other text that the reader can immediately access [1], [3]. Hypertext documents are interconnected by the hyperlinks. The hyperlinks are usually activated by a click which leads the user to the other text.
Multimedia
Multimedia represents any information that relates to the human's five senses [1]. This information is usually visual, audio, or textual. The information must be digitalized for a computer to be able to use it and it is then presented to viewers.
CLI
CLI, which stands for Command-Line Interface, is a text-based interface, one of the earlier forms of interaction with computers [1]. A user interacts with a computer by inputting lines of text, known as commands. It is not user-friendly nor intuitive as the commands must be memorized by the user.
GUI
GUI, which stands for Graphical User Interface, is a visual interface and one of the ways of interaction with computers [1], [4]. A user interacts with a computer through graphical elements such as icons, buttons, and similar. It is user-friendly and intuitive.
HCI
HCI, which stands for Human-Computer Interaction, is a field of study that focuses on the interaction between humans and computers [5]. Its goal is to improve design, usability, user experience, intuitiveness, and accessibility for users.
SFTP
SFTP, which stands for Secure File Transfer Protocol, is a secure, encrypted network protocol that is used for transferring files over a network [6]. It is an extension of the SSH protocol and because of its high reliability and security, it is considered a replacement for the FTP protocol [7].
SSH
SSH, which stands for Secure Shell or Secure Socket Shell, is a network protocol used to securely access and manage a device or server over an unsecured network, such as the Internet [8]. SSH is a replacement for Telnet as it is more reliable and has enhanced security features such as encryption.
Telnet
Telnet, which stands for TELecommunication NETwork, is a network protocol used to access and manage a remote device or server [9]. Telnet is outdated and insecure because it lacks fundamental security measures such as encryption and secure authentication. Telnet is widely replaced with SSH, a more secure alternative.
HTTP
HTTP, which stands for HyperText Transfer Protocol, is an application-layer protocol [10]. It is the foundation for data communication as it governs communication between web browsers and web servers [1].
HTTPS
HTTPS, which stands for HyperText Transfer Protocol Secure, is an extension of the HTTP protocol [1], [11]. It is a secure version of the HTTP protocol as it encrypts the data exchanged between web browsers and web servers.
HTML
HTML, which stands for Hypertext Markup Language, is the standard markup language for text files intended for display on a Web page within a Web browser [12]. The language is a set of symbols or codes that change the appearance and modify the text file into a structured document ready for display on the Web.
CSS
CSS, which stands for Cascading Style Sheets, is the standard style sheet language used to improve the presentation of a document written in a markup language [13]. It is used to create a style, format, and layout of the document.
TCP
TCP, which stands for Transmission Control Protocol, is one of the main protocols [1]. It is a communication protocol that enables message exchange over a network.
IP
IP, which stands for Internet Protocol, is the address system of the Internet, and it is fundamental for delivering information from the source to the target [1].
TCP/IP
TCP/IP, which stands for Transmission Control Protocol/Internet Protocol or simply Internet Protocol Suite, is a combination protocol created by TCP and IP protocols where IP protocol deals with routing and addressing the packets of information, while the TCP protocol ensures delivery [1].
